## Code Components

### CodeBlock

The `CodeBlock` component provides syntax highlighting with copy functionality and language detection:

<CodeBlock
  code={`function HelloWorld() {
    const message = "Hello, Luma Docs!";
    return <h1>{message}</h1>;
}`}
  language="tsx"
  title="Hello World Component"
/>

### Inline Code

Use the <Code>InlineCode</Code> component for short code snippets within text. You can also use <Kbd>Ctrl</Kbd> + <Kbd>C</Kbd> to show keyboard shortcuts.

### Code Tabs

The `CodeTabs` component is perfect for showing the same functionality in different languages:

<CodeTabs
  tabs={[
    {
      label: "React",
      language: "tsx",
      code: `import React from 'react';

export function Button({ children, onClick }) {
  return (
    <button 
      onClick={onClick}
      className="px-4 py-2 bg-blue-500 text-white rounded"
    >
      {children}
    </button>
  );
}`
    },
    {
      label: "Vue",
      language: "vue",
      code: `<template>
  <button 
    @click="onClick"
    class="px-4 py-2 bg-blue-500 text-white rounded"
  >
    <slot />
  </button>
</template>

<script>
export default {
  props: ['onClick']
}
</script>`
    },
    {
      label: "Svelte",
      language: "svelte",
      code: `<script>
  export let onClick;
</script>

<button 
  on:click={onClick}
  class="px-4 py-2 bg-blue-500 text-white rounded"
>
  <slot />
</button>`
    }
  ]}
  groupId="button-examples"
/>

## Interactive Examples

### Combining Components

You can combine multiple components to create rich, interactive documentation:

<Collapsible title="Complete API Example" intent="info" defaultOpen>
  <Callout type="tip" title="Best Practice">
    Always validate your API inputs and handle errors gracefully.
  </Callout>

  <CodeTabs
    tabs={[
      {
        label: "TypeScript",
        language: "typescript",
        code: `interface User {
  id: string;
  name: string;
  email: string;
}

async function fetchUser(id: string): Promise<User> {
  try {
    const response = await fetch(\`/api/users/\${id}\`);
    if (!response.ok) {
      throw new Error('Failed to fetch user');
    }
    return await response.json();
  } catch (error) {
    console.error('Error fetching user:', error);
    throw error;
  }
}`
      },
      {
        label: "JavaScript",
        language: "javascript", 
        code: `async function fetchUser(id) {
  try {
    const response = await fetch(\`/api/users/\${id}\`);
    if (!response.ok) {
      throw new Error('Failed to fetch user');
    }
    return await response.json();
  } catch (error) {
    console.error('Error fetching user:', error);
    throw error;
  }
}`
      }
    ]}
    groupId="api-example"
  />

  <Callout type="warning">
    Remember to handle network errors and validate the response data before using it in your application.
  </Callout>
</Collapsible>
